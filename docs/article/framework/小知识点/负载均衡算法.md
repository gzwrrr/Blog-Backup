# 负载均衡算法

1. 轮询法：平衡地将请求轮流到每一台服务器上
2. 加权轮询法：根据不同权重请求服务器
3. 随机法：通过随机算法分配
4. 加权随机法：与加权轮询法相似
5. 源地址哈希法：根据客户端 IP 取哈希值再取模得出目标服务器
6. 最小连接数法：动态选择请求数或者请求压力最小的机器进行请求
