# Redis

:::info 什么是 Redis

Redis，**Remote Dictionary Server（远程数据服务）**，是一种支持key-value等多种数据结构的存储系统。可用于缓存，事件发布或订阅，高速队列等场景。支持网络，提供字符串，哈希，列表，队列，集合结构直接存取，基于内存，可持久化。
